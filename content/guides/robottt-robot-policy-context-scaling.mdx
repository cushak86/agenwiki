| --- | --- |
| 학습 문맥 길이 | 8,000 타임스텝 (기존 대비 1,000배) |
| 실기 성능 개선 | 단일 스텝 베이스라인 대비 +87% |
| 문맥 길이 효과 | 1K 대비 8K 학습 시 +62% |
| 검증 과제 | 5분 길이 10단계 조립 작업 |

로봇 파운데이션 모델은 그동안 한 번에 한 스텝, 혹은 아주 짧은 과거만 참고해 다음 행동을 결정해 왔다. 엔비디아 연구팀이 공개한 RoboTTT(Test-Time-Training Robot Policies)는 이 시각-운동 문맥을 8,000 타임스텝까지 늘리면서도 추론 지연을 키우지 않는 방법을 제시한다.

문맥을 이렇게 늘리는 게 왜 중요한가. 사람의 시연 영상을 한 번 보여주는 것만으로 로봇이 그 자리에서 행동을 모방하거나(one-shot in-context imitation), 실행 중에 정책을 즉석으로 개선하거나, 방해 요인에 더 잘 버티는 능력이 이 지점에서 처음으로 열린다는 것이 연구팀의 주장이다. 특히 여러 단계로 이어지는 긴 시간대 작업에서 이 차이가 두드러진다.

## 1. 무엇을 바꿨나 — 시각-운동 문맥의 스케일

기존 [비전-언어-액션(VLA)](/glossary/multimodal) 정책들은 짧은 과거 정보만으로 다음 행동을 예측하는 구조였다. RoboTTT는 이 구조에 테스트타임 트레이닝(Test-Time Training)을 결합해, 순환 상태(recurrent state)를 "빠른 가중치(fast weights)"로 구성한다. 이 가중치는 학습 중은 물론 추론 중에도 경사하강으로 계속 업데이트되며, 과거 이력을 가중치 공간으로 압축해 저장하고 필요할 때 꺼내 쓰는 방식으로 긴 문맥을 다룬다.

## 2. 얼마나 벌어지나 — 문맥 길이별 성능 비교

연구팀은 문맥 길이를 늘렸을 때 성능이 어떻게 달라지는지를 정면으로 측정했다.

| 비교 축 | 결과 |
| --- | --- |
| 단일 스텝 컨텍스트 vs RoboTTT | 실기 조작 종합 성능 +87% |
| 1K 타임스텝 사전학습 vs 8K 타임스텝 사전학습 | +62% |
| 5분·10단계 조립 과제 완주 | 베이스라인은 전무, RoboTTT는 완주 |

주목할 점은 사전학습 문맥 길이를 늘릴수록 폐루프(closed-loop) 성능이 꾸준히 개선되는 경향을 처음으로 관찰했다는 것이다. 이는 문맥 길이 자체가 로봇 파운데이션 모델의 새로운 스케일링 축이 될 수 있음을 시사한다.

> **요약**: 문맥을 8,000 타임스텝으로 늘리자 실기 조작 성능이 87% 개선됐고, 기존엔 아무도 끝내지 못한 5분짜리 10단계 조립 작업을 완주했다.

## 3. 어떻게 학습시켰나 — 시퀀스 액션 포싱과 절단 역전파

문맥 길이를 8K까지 늘리려면 학습 자체를 감당할 수 있어야 한다. RoboTTT는 시퀀스 액션 포싱(sequence action forcing)과 시간에 따른 절단 역전파(truncated backpropagation through time)를 결합해 이 학습 레시피를 완성했다. 핵심은 추론 시에도 빠른 가중치가 계속 업데이트된다는 점으로, 이 때문에 문맥이 길어져도 추론 지연이 늘어나지 않는다고 연구팀은 설명한다.

## 4. 실무 의미 — 로봇 정책 설계자가 볼 것

이 결과는 [LLM](/glossary/llm)에서 이미 익숙한 "긴 컨텍스트 윈도"의 이점이 로봇 정책에도 적용될 수 있음을 보여준다. 다만 저자들이 검증한 것은 특정 실기 조작 과제(조립 등)에 한정되며, 다른 로봇 형태나 작업 유형까지 동일하게 일반화되는지는 논문 초록만으로는 확인할 수 없다.

> **주의**: 87%, 62%라는 개선폭은 논문이 제시한 특정 벤치마크·베이스라인 기준이다. 다른 작업 환경에서의 재현 여부는 별도 검증이 필요하다.

## 마무리 — agenwiki의 한줄평

로봇 정책에서 "문맥 길이"가 새로운 스케일링 축이 될 수 있다는 첫 관찰은, 시각-언어 모델의 성장 서사가 로보틱스로도 옮겨붙고 있음을 보여주는 신호다. 다만 추론 지연 없이 8K 문맥을 유지한다는 주장이 다양한 실기 환경에서도 유지되는지는 후속 검증을 지켜볼 필요가 있다.
