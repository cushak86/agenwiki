로봇 [파운데이션 모델](/glossary/foundation-model)은 대개 한 스텝, 길어야 몇 스텝 분량의 시각-운동 정보만 보고 다음 행동을 정한다. 기억할 수 있는 과거가 짧다는 뜻이다. NVIDIA, 스탠퍼드대 등이 참여한 이번 연구는 이 문맥 범위를 기존 최고 수준보다 세 자릿수 배 넓혀, 추론 지연을 늘리지 않고도 8000스텝까지 확장한 로봇 정책 RoboTTT를 제시한다.

이렇게 문맥을 늘리면 무엇이 달라지는가가 이 논문의 핵심 질문이다. 연구진은 사람 시연 영상을 한 번 보여주는 것만으로 곧바로 따라 하는 원샷 모방, 실행 중 즉석 정책 개선, 외란에 대한 강건성, 다단계 장기 과제에서의 성능 향상을 확인했다고 보고한다. 사전학습 문맥 길이가 늘수록 폐루프 성능이 꾸준히 좋아지는 현상을 관찰한 것도 이번이 처음이라고 밝힌다.

| 항목 | 수치 |
| --- | --- |
| 확장 문맥 길이 | 8000스텝(기존 대비 1000배) |
| 단일 스텝 기준선 대비 | 성능 87% 향상 |
| 1000스텝 사전학습 대비 | 성능 62% 향상 |
| 완주한 과제 | 5분·10단계 조립(기준선은 전무) |

## 1. 무엇이 문제인가 — 짧은 기억을 가진 로봇 정책

기존 [비전-언어-행동](/glossary/vision-language-action) 정책 다수는 단일 스텝이나 짧은 과거 이력만 보고 다음 동작을 결정한다. 문맥이 짧으면 사람이 시범을 보여줘도 그 흐름 전체를 참고하기 어렵고, 여러 단계가 이어지는 긴 과제에서는 앞선 실수나 성공의 맥락을 놓치기 쉽다. 연구진은 이 한계를 정면으로 다뤄, 문맥 길이 자체를 로봇 정책의 새로운 확장 축으로 세운다.

## 2. 얼마나 벌어지나 (비교)

| 비교 대상 | 결과 |
| --- | --- |
| 단일 스텝 기준선 vs RoboTTT | 87% 성능 향상 |
| 1000스텝 사전학습 vs 8000스텝 사전학습 | 62% 성능 향상 |
| 5분·10단계 조립 과제 완주 | 기준선 모델은 단 한 번도 성공 못함, RoboTTT는 완주 |

> **요약**: 문맥을 8000스텝까지 늘렸을 때 성능이 선형이 아니라 꾸준히, 그리고 큰 폭으로 좋아졌다는 점이 이번 결과의 핵심이다.

## 3. 어떻게 작동하나 — 테스트 타임 트레이닝과 고속 가중치

RoboTTT의 핵심은 테스트 타임 트레이닝(Test-Time Training)을 로봇 파운데이션 모델에 결합한 것이다. 훈련과 추론 양쪽에서 경사하강법으로 갱신되는 "고속 가중치(fast weights)"를 순환 상태로 사용해, 과거 이력을 가중치 공간에 압축하고 필요할 때 그 안에서 문맥 정보를 꺼내 쓰는 구조다. 긴 이력을 그대로 들고 다니는 대신 압축된 형태로 저장하기 때문에 문맥을 크게 늘려도 추론 지연이 늘지 않는다고 설명한다.

훈련 단계에서 이 긴 문맥을 실제로 학습시키기 위해, 연구진은 시퀀스 액션 포싱과 절단된 시간 역전파(truncated backpropagation through time)를 결합한 학습 레시피를 사용했다.

> **주의**: 이번 결과는 특정 실환경 로봇 조작 과제를 기준으로 한 것이며, 다른 로봇 형태나 과제군에서도 동일한 폭의 개선이 재현되는지는 이 논문만으로 단정하기 어렵다.

## 4. 실무 의미 — 문맥 길이라는 새로운 확장 축

연구진은 8000스텝 문맥으로 사전학습한 모델이 1000스텝으로 사전학습한 동일 모델보다 62% 더 나은 성능을 보였다는 점에서, 문맥 길이를 모델 크기나 데이터양과 나란한 로봇 파운데이션 모델의 새로운 스케일링 축으로 제안한다. 다섯 개 정거장에 걸친 5분짜리 조립 과제를 끝까지 완수한 것도 RoboTTT가 유일했다.

> **팁**: 로봇 정책을 설계·평가하는 팀이라면 모델 파라미터 수뿐 아니라 훈련·추론에 사용하는 문맥 길이도 별도의 확장 변수로 관리할 필요가 있다.

## 마무리 — agenwiki의 한줄평

로봇 정책의 "기억력"을 세 자릿수 배 늘리는 접근이 실제 조작 과제에서 87%라는 뚜렷한 성능 차이로 이어졌다는 점이 눈에 띈다. 파라미터 확장 중심이던 로봇 파운데이션 모델 경쟁에 문맥 길이라는 새로운 변수가 더해질 가능성을 보여준다.
